# GaCha Pong 💊✨
![gacha](https://github.com/chachohee/spboot_project/assets/83406032/629fb56d-ee1b-407c-8d20-bc861de06545)
랜덤으로 이미지를 얻고, 갤러리에 자신이 뽑은 이미지를 자랑할 수 있습니다!😊<br>
게시판 CRUD를 중점으로 한 스프링부트 프로젝트입니다.
